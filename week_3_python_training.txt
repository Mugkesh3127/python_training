1.Implement the Two Sum problem: Given nums=[2,7,11,15] and target=9, return indices of two numbers that add up to target.
What is your solution's time and space complexity?

	def two_sum(nums, target):
		seen = {}

		for i, num in enumerate(nums):
			complement = target - num

			if complement in seen:
				return [seen[complement], i]

			seen[num] = i

		return []

	# Example
	nums = [2, 7, 11, 15]
	target = 9

	print(two_sum(nums, target))
	# Output: [0, 1]
	
2.Write a function that finds the longest substring without repeating characters. Example: 'abcabcbb' → 3 ('abc'). Use the sliding window technique.

	def length_of_longest_substring(s):
		seen = set()
		left = 0
		max_length = 0

		for right in range(len(s)):
			while s[right] in seen:
				seen.remove(s[left])
				left += 1

			seen.add(s[right])
			max_length = max(max_length, right - left + 1)

		return max_length

	# Example
	print(length_of_longest_substring("abcabcbb"))  # 3
	print(length_of_longest_substring("bbbbb"))     # 1
	print(length_of_longest_substring("pwwkew"))    # 3
	
3.Explain the two-pointer technique. Write a function that removes duplicates from a sorted array in-place and returns the new length.

	The two-pointer technique uses two indices (pointers) to traverse a data structure, usually an array or string. It helps solve problems efficiently by avoiding nested loops.

	Common uses:

		Removing duplicates from a sorted array
		Finding pairs with a target sum
		Reversing arrays/strings
		Palindrome checking
		Removing Duplicates from a Sorted Array (In-Place)

	Since the array is sorted, duplicates are adjacent. We can use:

		slow pointer: position to place the next unique element
		fast pointer: scans through the array.
		
	def remove_duplicates(nums):
		if not nums:
			return 0

		slow = 1

		for fast in range(1, len(nums)):
			if nums[fast] != nums[fast - 1]:
				nums[slow] = nums[fast]
				slow += 1

		return slow
		
	nums = [1, 1, 2, 2, 3, 4, 4]

	slow = 1

	fast=1 -> duplicate (1), skip
	fast=2 -> unique (2), place at nums[1]
	fast=3 -> duplicate (2), skip
	fast=4 -> unique (3), place at nums[2]
	fast=5 -> unique (4), place at nums[3]
	fast=6 -> duplicate (4), skip

	Result: [1, 2, 3, 4, ...]
	Length = 4
	
	Complexity Analysis:
		Time Complexity: O(n)
		Each element is visited once.
		Space Complexity: O(1)
		The array is modified in place without using extra data structures.
		
4.Given an array [1,-2,3,4,-1,2,1,-5,4], find the contiguous subarray with the largest sum (Kadane's algorithm). What is the time complexity?

	def max_subarray(arr):
		current_sum = max_sum = arr[0]

		for num in arr[1:]:
			current_sum = max(num, current_sum + num)
			max_sum = max(max_sum, current_sum)

		return max_sum

	print(max_subarray(arr))  # 9
	
	The contiguous subarray with the largest sum is:

	[3, 4, -1, 2, 1]

	Sum:

	3 + 4 - 1 + 2 + 1 = 9
	
	Complexity Analysis:
		Time Complexity: O(n)
		The array is traversed only once.
		Space Complexity: O(1)
		Only a few variables (current_sum, max_sum) are used.
		
5.Write a function to determine if two strings are anagrams of each other. Provide two different approaches and compare their complexities.

	Approach 1: Sort Both Strings

		If two strings are anagrams, their sorted characters will be identical.

		def is_anagram_sort(s1, s2):
			return sorted(s1) == sorted(s2)

		# Example
			print(is_anagram_sort("listen", "silent"))  # True
			
		Complexity
			Time Complexity: O(n log n)
				Sorting each string takes O(n log n).
			Space Complexity: O(n)
				Sorted copies of the strings are created.
		
	Approach 2: Use a Counter (Hash Map)

		Count the frequency of each character in both strings and compare the counts.

		from collections import Counter

		def is_anagram_counter(s1, s2):
			return Counter(s1) == Counter(s2)

		# Example
		print(is_anagram_counter("listen", "silent"))  # True
		
		Complexity:
			Time Complexity: O(n)
				Each character is counted once.
			Space Complexity: O(k)
				Where k is the number of unique characters (O(n) in the worst case).
				
		Comparison:
			Approach	Time Complexity	Space Complexity	Notes
			Sorting	O(n log n)	O(n)	Simple and easy to understand
			Counter / Hash Map	O(n)	O(k)	More efficient for large strings
			
			Example
				print(is_anagram_sort("race", "care"))      # True
				print(is_anagram_counter("race", "care"))   # True

				print(is_anagram_sort("hello", "world"))    # False
				print(is_anagram_counter("hello", "world")) # False
				
			For production code, the Counter/hash map approach is usually preferred because it runs in O(n) time,
			compared to O(n log n) for the sorting approach.
			
6.Implement a function that rotates an array by k positions to the right.
Example: [1,2,3,4,5] rotated by 2 → [4,5,1,2,3]. Can you do it in O(1) extra space?

	Solution (O(1) Extra Space)
	def rotate(nums, k):
		n = len(nums)
		k %= n

		def reverse(left, right):
			while left < right:
				nums[left], nums[right] = nums[right], nums[left]
				left += 1
				right -= 1

		# Reverse the entire array
		reverse(0, n - 1)

		# Reverse the first k elements
		reverse(0, k - 1)

		# Reverse the remaining elements
		reverse(k, n - 1)

		return nums

	# Example
		nums = [1, 2, 3, 4, 5]
		
	print(rotate(nums, 2))
	# Output: [4, 5, 1, 2, 3]
	
	How It Works
	For nums = [1,2,3,4,5] and k = 2:

	Original:      [1, 2, 3, 4, 5]
	Reverse all:   [5, 4, 3, 2, 1]
	Reverse first 2:[4, 5, 3, 2, 1]
	Reverse rest:[4, 5, 1, 2, 3]
				   
	Complexity Analysis
		Time Complexity: O(n)
			Each element is moved a constant number of times.
		Space Complexity: O(1)
			Rotation is performed in-place using only a few variables.
			
	Alternative (Simpler, but not O(1) Space)
	def rotate(nums, k):
		k %= len(nums)
		return nums[-k:] + nums[:-k]
	Time Complexity: O(n)
	Space Complexity: O(n)

	The in-place reversal algorithm rotates the array in O(n) time and O(1) extra space, producing:
	[1, 2, 3, 4, 5] -> [4, 5, 1, 2, 3].
	
7.Best Time to Buy and Sell Stock: Given prices=[7,1,5,3,6,4],
find the maximum profit from one buy and one sell. Write the solution and explain your approach.

	Solution
	def max_profit(prices):
		min_price = float('inf')
		max_profit = 0

		for price in prices:
			min_price = min(min_price, price)
			profit = price - min_price
			max_profit = max(max_profit, profit)

		return max_profit

	# Example
	prices = [7, 1, 5, 3, 6, 4]
	print(max_profit(prices))  # 5
	
	Approach

	Use a single pass through the array:
		1.Keep track of the lowest price seen so far (min_price).
		2.For each price, calculate the profit if you sold on that day:
			profit = current_price - min_price
		3.Update the maximum profit whenever a larger profit is found.
		
	Walkthrough
		Prices = [7, 1, 5, 3, 6, 4]

		Day 1: price=7
		min_price=7
		profit=0
		max_profit=0

		Day 2: price=1
		min_price=1
		profit=0
		max_profit=0

		Day 3: price=5
		profit=5-1=4
		max_profit=4

		Day 4: price=3
		profit=3-1=2
		max_profit=4

		Day 5: price=6
		profit=6-1=5
		max_profit=5

		Day 6: price=4
		profit=4-1=3
		max_profit=5
		
	Result
		Buy at price = 1
		Sell at price = 6
		Maximum Profit = 5
		
	Complexity Analysis
		Time Complexity: O(n)
			Only one pass through the array.
		Space Complexity: O(1)
			Uses only a few variables.
			

	For:

		prices = [7, 1, 5, 3, 6, 4]

	Maximum Profit = 5 (buy at 1, sell at 6).

		Time Complexity: O(n)
		Space Complexity: O(1)
		
8.Write a function that, given a string of parentheses and brackets, checks if it is balanced. Example: '([])' → True, '([)]' → False.

	Solution Using a Stack
		def is_balanced(s):
			stack = []
			pairs = {
				')': '(',
				']': '[',
				'}': '{'
			}

			for char in s:
				if char in '([{':
					stack.append(char)
				elif char in ')]}':
					if not stack or stack.pop() != pairs[char]:
						return False

			return len(stack) == 0

		# Examples
		print(is_balanced("([])"))   # True
		print(is_balanced("([)]"))   # False
		print(is_balanced("{[()]}")) # True
		print(is_balanced("((("))    # False
	
	How It Works
		Use a stack to keep track of opening brackets.
		When an opening bracket ((, [, {) is encountered, push it onto the stack.
		When a closing bracket (), ], }) is encountered:
			Check if the stack is empty.
			Pop the top element and verify it matches the corresponding opening bracket.
		After processing the string, the stack must be empty for the string to be balanced.
		
	Example: "([])"
		(  -> push (
		[  -> push [
		]  -> pop [, matches
		)  -> pop (, matches

		Stack is empty -> True
		
	Example: "([)]"
		(  -> push (
		[  -> push [
		)  -> expected (, found [

		Mismatch -> False
		
	Complexity Analysis
		Time Complexity: O(n)
			Each character is processed once.
		Space Complexity: O(n)
			In the worst case, all opening brackets are stored in the stack.
			
	"([])" → True
	"([)]" → False

	Approach: Use a stack to match opening and closing brackets in O(n) time.
